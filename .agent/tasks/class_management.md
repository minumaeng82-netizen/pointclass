## 학급 생성 및 관리 기능 (Class Management)

선생님 대시보드에 **학급 관리(설정)** 탭을 추가하고, 새로운 학급을 생성하고 학생 명부를 관리하는 기능을 구현했습니다.

### 주요 기능
1.  **학급 생성 마법사**:
    *   **1단계**: 학년(3~6학년), 반(1~12반), 학생 수(1~50명)를 입력합니다.
    *   **2단계**: 입력한 학생 수에 맞춰 번호와 기본 이름("학생 N")이 자동 생성됩니다. 선생님은 여기서 학생들의 실명을 바로 입력할 수 있습니다.
2.  **학급 관리**: 생성된 학급 목록을 카드로 확인하고, 필요한 경우 삭제할 수 있습니다.
3.  **학생 로그인 지원**: 생성된 학생들은 `0000` (초기 비밀번호)으로 로그인할 수 있으며, 로그인 후 비밀번호를 변경하도록 안내받습니다.

### UI 디자인
- 선생님 대시보드 사이드바 하단에 **설정(Settings)** 탭을 추가했습니다.
- 직관적인 모달 인터페이스를 통해 복잡한 과정 없이 학급을 세팅할 수 있습니다.

### 사용 방법
1.  선생님 대시보드 로그인 (대시보드 하단 텍스트 클릭)
2.  사이드바 맨 아래 **설정** 탭 클릭
3.  우측 상단 **+ 학급 생성** 버튼 클릭
4.  정보 입력 후 **다음** → 학생 이름 확인/수정 후 **생성 완료**

이제 선생님은 각 반별로 데이터를 독립적으로 관리할 수 있습니다!
